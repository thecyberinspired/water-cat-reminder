# Water Cat Reminder 🐱💧

A tiny desktop reminder with a pixel cat that pops up every hour.

It shows a cute animated cat and a small message like:

- drink water ♡
- mon chouchou ♡
- prends soin de toi ♡

## Features

- animated pixel cat (GIF)
- random cute messages
- hourly reminder
- small toast-style window
- transparent corners on Windows

## Run

```bash
python main.py

## Files
main.py   - reminder widget
